## About The Project

HIFI Movie API is a small server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

## Features

   <br>

#### FEATURE 1: RETURN A LIST OF ALL MOVIES TO THE USER

<br>

#### FEATURE 2: RETURN DATA (DESCRIPTION, GENRE, DIRECTOR, IMAGE) ABOUT A SINGLE MOVIE BY TITLE TO THE USER

<br>

#### FEATURE 3: RETURN DATA ABOUT A GENRE (DESCRIPTION) BY NAME/TITLE (EG. 'THRILLER')

<br>

#### FEATURE 4: RETURN DATA ABOUT A DIRECTOR (NAME, BIO)

<br>

#### FEATURE 5: ALLOW NEW USER TO REGISTER

<br>

#### FEATURE 6: ALLOW USERS TO UPDATE THEIR USER INFO (USERNAME, PASSWORD, EMAIL, DATE OF BIRTH)

<br>

#### FEATURE 7: ALLOW USERS TO ADD A MOVIE TO THEIR LIST OF FAVORITES

<br>

#### FEATURE 8: ALLOW USERS TO REMOVE A MOVIE FROM THEIR LIST OF FAVORITES

<br>

#### FEATURE 9: ALLOW EXISTING USER TO DEREGISTER

<br>
